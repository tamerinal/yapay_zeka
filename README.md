## CSV Yükleme Arayüzü

`load_csv.py` dosyası, yerel diskinizden CSV dosyası seçmenizi sağlayan basit bir Tkinter arayüzü sunar.
Çalıştırmak için:

```bash
python load_csv.py
```

Açılan pencereden **CSV Dosyası Yükle** butonuna basarak dosyayı seçebilirsiniz. Seçilen dosya yolu ve ilk satırlar ekranda görüntülenir.
