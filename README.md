## LLM Upload Interface

The `front_end/llm_upload` directory contains a simple Flask-based web
application for uploading LLM model files. To run the interface:

```bash
cd front_end/llm_upload
pip install -r requirements.txt
python app.py
```

The server will start on `http://127.0.0.1:5000/` where you can upload
your model files.
